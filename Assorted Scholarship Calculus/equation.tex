\textbf{Equation Problem}

Find all real values of $ \alpha $ which satisfy the following equation:

\begin{displaymath}
  \left( \alpha - \frac {1}{2} \right) \left( \log_{0.5} \alpha \right) ^2 + \log_{0.5} \sqrt{\alpha^{5-10\alpha}} + 6\alpha - 3 = 0
\end{displaymath}
